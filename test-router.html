Debugging Steps

  1. Check Frontend Server: Make sure React dev server is running on port 3000
  2. Check Backend Server: Make sure Node.js server is running on port 5000
  3. Check Browser Console: Open F12 and look for JavaScript errors
  4. Check Network Tab: Look for failed requests
  5. Clear Browser Cache: Try Ctrl+F5 to hard refresh
  6. Check LocalStorage: Clear any existing tokens
